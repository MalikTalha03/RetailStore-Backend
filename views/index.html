Please use below routes to access the API

Public Routes

  • Login
    • POST /login
  • Register
    • POST /register

Protected Routes

  • Products
    • GET /products
    • GET /products/:id
    • POST /products
    • PUT /products/:id
    • DELETE /products/:id
  • Category
    • GET /category
    • GET /category/:id
    • POST /category
    • PUT /category/:id
    • DELETE /category/:id
  • Employee
    • GET /employee
    • GET /employee/:id
    • POST /employee
    • PUT /employee/:id
    • DELETE /employee/:id
  • Credit Customers
    • GET /creditcustomer
    • GET /creditcustomer/:id
    • POST /creditcustomer
    • PUT /creditcustomer/:id
    • DELETE /creditcustomer/:id
  • Customer
    • GET /customer
    • POST /customer
    • PATCH /customer/:id/orders
    • PATCH /customer/:id/orders/:orderId/details
    • PATCH /customer/:id/orders/:orderId/transactions
  • Supplier
    • GET /supplier
    • POST /supplier
    • PATCH /supplier/:id/orders
    • PATCH /supplier/:id/orders/:orderId/details
    • PATCH /supplier/:id/orders/:orderId/transactions
